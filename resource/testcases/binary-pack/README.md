# 文件读取提示

---
每个文件夹下有三个文件：`test.in` `outp.out` `time.out`

## test.in

数据输入文件，格式为：

```
n
0 v0 c0
1 v1 c1
...
c
```

表示有`n`个物品，容量上限为`c`，第i个物品的价值为`vi`，体积为`ci`。

## outp.out

近似最优解文件，格式为：

```
max_value
v0 c0
v0 c0
...
```

表示取得的最大价值为`max_value`，选择的第i个物品的价值为`vi`，体积为`ci`（与原始索引无关）。
